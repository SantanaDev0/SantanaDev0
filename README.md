# SantanaDev0

Portfólio pessoal customizado com tema de anime animado.

## Tecnologias
- React
- TypeScript
- Vite
- TailwindCSS
- Framer Motion
- GSAP

## Instalação

```bash
npm install
npm run dev
```

## Deploy
O projeto está configurado para deploy na Vercel.
